My Work

Data2Linear App

This project was created by Diego Maldonado and Daniel Canhedo for the 'CuseHacks 2024 24hr hackathon project. The project's goal is to provide a web interface where a user can upload a CSV file, store its data in a SQLite database, and create a linear regression model based off of two provided variables.

Happiness Simulation

This Python project simulates decision-making strategies—exploration, exploitation, and epsilon-greedy—over 200 days to maximize cumulative happiness. By modeling outcomes with statistical functions and simulations, the program evaluates the effectiveness of each approach. This project highlights my skills in algorithm design, probabilistic modeling, and data analysis.

Cat Recognition
